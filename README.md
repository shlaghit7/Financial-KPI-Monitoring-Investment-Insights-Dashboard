# 📊 Financial KPI Monitoring & Investment Insights Dashboard

## 🚀 Project Overview

The **Financial KPI Monitoring & Investment Insights Dashboard** is a professional Tableau Business Intelligence project designed to monitor financial performance, evaluate investment efficiency, analyze portfolio growth, and support strategic decision-making.

This dashboard enables finance teams, investment analysts, CFOs, wealth managers, and business leaders to track critical financial KPIs such as Revenue, Profit, ROI, Profit Margin, Portfolio Value, Budget Variance, and Investment Performance across different regions, asset classes, and investment categories.

---

# 🎯 Project Objective

The primary objective of this project is to:

* Monitor financial performance in real-time
* Track key financial KPIs
* Analyze investment profitability
* Evaluate portfolio growth
* Compare Budget vs Actual performance
* Identify high-performing investment categories
* Support strategic business and investment decisions

# 🧮 Tableau Calculated Fields

### Total Profit

Profit = Revenue - Expenses

### Profit Margin %

Profit ÷ Revenue × 100

### ROI %

Profit ÷ Investment Amount × 100

### Budget Variance %

(Actual Revenue - Budgeted Revenue) ÷ Budgeted Revenue × 100

### Portfolio Growth %

(Current Portfolio Value - Previous Portfolio Value) ÷ Previous Portfolio Value × 100

### Investment Performance Score

Weighted score based on ROI and Profit Margin.
